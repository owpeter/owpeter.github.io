Git基础

1. 获取Git仓库

创建版本仓库

  • 在本地新建一个文件夹，编写好程序后通过 git init命令把这个文件夹变为仓库
  • 使用git remote add origin <url>，将本地仓库与远程仓库关联

克隆远程仓库

  • 使用git clone命令来克隆仓库中几乎全部数据。
  • 命令的格式为git clone [url]

远程仓库的使用

关联

在远程托管平台上创建好远程仓库后，使用git remote add <short_name> <url>命令，将本地仓库与远程仓库进行关联。

以后，可以使用字符串<short_name>代替url。

推送

使用git push <remote_name> <branch_name>，将本地仓库中的内容推送到<remote_name>仓库的<branch_name>分支中

需要注意的是，只有当文件被commit过后，才能push到远程仓库，而在暂存区与未暂存的文件都不能直接被push。

拉取

使用git pull <remote_name> <branch_name>命令来将远程仓库中的某一分支与本地分支进行合并
